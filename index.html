About Our Project

Helping our mentees expand their professional brand through web development!

For our LIT Creations project we wanted to expose our mentees to the foundations of web development.

We thought that the best method to help our mentees understand these concepts is if they worked on creating some type of websites.

Each mentee devised their own personal websites using HTML/CSS/Bootstrap. They also learned basic concepts of Git while committing their code to GitHub.

We would all typically meet up on Fridays to teach them the process that is taken in forming a website from ideation and prototyping, to creating the navigation bar, and finally ways to include the stylistic components.

The first portion of our meetings is for our mentees to showcase their updates (where the mentors provide feedback). From there, we will delve into a certain topic, and for the remainder of the time, they are able to practice or apply the knowledge the gained.

It is amazing to see the rapid growth of what Audrey and Chelsea learnt through LIT Creations and we hope that this project in creating a personal website will help them stand out in the pool and will be beneficial in showcasing who they are and landing several internships, co-ops, and scholarships/awards in the future. Tech Stack

What technologies did our mentees learn?

HTML

First our mentees learned the fundamentals of website development with basic HTML and CSS tags

CSS

Our mentees learned how to add styling to HTML elements using CSS

NPM

Our mentees installed and learned how to use Node Package Manager to install new libraries that they integrated into their code

Bootstrap 4

Our mentees used Bootstrap in their website to create modern and responsive user interface components

GitHub

Our mentees learned the fundamentals of Git and GitHub to collaborate on code throughout the semester

Web Hosting

To host our website on the Internet, our mentees learned how to integrate code with the free hosting service GitHub Pages

Project Timeline



  • Milestone #1

    Learning new technologies

    We taught our mentees the fundamentals of website design and development. We introduced them to HTML, and walked them through tutorials for getting started.

  • Milestone #2

    Starting to code

    Our mentees began coding their own websites based on what they learned! We started with a simple Bootstrap Navigation Bar, and continued to add more components.

  • Milestone #3

    Improving the User Interface

    Our mentees learned how to use CSS to improve the styles on their websites. This included adding background images, custom fonts and colors, and improved layouts.

  • Milestone #4

    Combining our work

    We ended by connecting Audrey and Chelsea's finished portfolios to the common page developed by the mentors. Scroll down to see the result!
